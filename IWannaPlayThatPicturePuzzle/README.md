# iwptpp.asl


## 主な機能
「今日は左と上の数字を見てマス目を塗りつぶすアレを無限にやりたい！」用の Autoplitter です。

### N x N
|自動操作|説明|
|:--|:--|
|自動スタート|1. 初期設定画面で決定を選択した瞬間<br>2. 事前に問題を生成後に開始を選択した瞬間|
|自動リセット|タイトル画面でゲームスタートを選択した瞬間|
|自動ラップ|今日解いた数が増加した瞬間|

### 設定項目
- スタートタイミング
  - 初回の問題生成をタイムに含めるか否か
- 自動ラップの間隔
  - 1問ごと
  - 5問ごと
  - 10問ごと

### IGT 計測
ゲーム内の合計時間を取得し、LivesplitのGame Timeを同期します。

### 対応バージョン
- ver. 1.8.2
- ver. 1.7.0-1.7.7
- ver. 1.6.x
- ver. 1.5.3


## Autosplitter の使用方法

### 設定手順
1. Splits Editor の Game Name に「I Wanna Play That Picture Puzzle with Numbers All Day!」と入力します。
1. Activate ボタンを押します。
1. 必要に応じて Settings ボタンを押して設定を行います。

### タイムの精査を推奨
ルール通りのタイミングでのタイマー操作は保証していないため、タイムの精査を推奨しています。

この Autosplitter は 60 Hz でゲームの内部値を監視することでタイマー操作タイミングを判定しています。<br>
ゲームの内部値が変わったタイミングとルール上のスタート and or ストップのタイミングとが厳密に一致するとは限らないため、ルール通りのタイミングでタイマー操作することは保証できません。

### Autosplitter の重複に注意
Autosplitter は Splits Editor から使用する上記の方法と、Layout Editor から使用する方法とがありますが、一度に使用する Autosplitter は1つのみとしてください。


## 既知の不具合・仕様
- サイズを変更して次の問題へ行った場合、合計時間がリセットされるゲーム側の仕様によりLivesplit側で計測しているIGTがリセットされたり正しくないタイムになる場合があります。
- ゲームの更新が頻繁に行われており、asl製作者がゲームの更新に気付かない事があるかもしれません。そのような場合にお声がけいただけると助かります。
- ゲームの更新頻度についていけなくなった場合はごめんなさい。
